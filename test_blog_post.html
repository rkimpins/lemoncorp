Metrics, or how we measure distance

Rigorous Definition

So what is a metric? Let's start with the axiomatic definition, and build from there

From Wikipedia:
A metric on a set X is a function (called distance function or simply distance) d : X × X → [0, ∞) where [0, ∞) is the set of non-negative real numbers and for all x , y , z ∈ X the following three axioms are satisfied:
  1. d(x, y) = 0 ⇔ x = y identity of indiscernibles
  2. d(x, y) = d(y, x) symmetry
  3. d(x, y) ≤ d(x, z) + d(z, y) triangle inequality

What does this mean?

Identity of indescernibles

If two points are 0 distance apart, they are in fact the same point. Also, if two points are equal, then they must be 0 distance apart

symmetry

The distance from our first point to our second point must be equal to the distance from our second point to our first point. Distance doesn't change when we swap the direction we are going in

triangle inequality

If we are travelling from point A to point B, we have a certain distance between them. If we instead choose to take a detour and go from point A to point C and then from point C to point B, the distance must be greater than or equal to our original distance.

Euclidean distance / Straight-line distance

Euclidean distance is what most people think of when they think about measuring distance. It is called straight-line distance because you can imagine place a ruler between the two points and using that as the measure of distance.
[PICTURE OF 2 BY 1 GRAPH]
In the graph above, if we measure the length of the line between the points with a ruler, we would get the euclidean distance. In this case, it is \sqrt{5}. How did we get this though? Well, we use our old friend a^2 + b^2 = c^2 a^2 + b^2 = c^2 -> 1^2 + 2^2 = c^2 -> 5 = c^2 -> c = \sqrt{5} The general formula given two points A and B is:

d(Point_1, Point_2) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}

A brief note on higher dimensions

We won't always be working in 2D. It is pretty common to need to find distances in 3D, and we can actually extend the definition of a metric to higher dimensions. For the Euclidean distance, we get something like the following formula

d(p,q) = \sqrt{\sum^n_{i=0} (p_i - q_i)^2}
Now you can find the euclidean metric in an n-dimensional cartesian space. Is this helpful? Not sure, but it sounds impressive. -\_o<
